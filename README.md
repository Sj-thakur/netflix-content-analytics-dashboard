# Netflix Content Analytics Dashboard

End-to-end Netflix content analytics project using Python, SQL, and Power BI.

## Project Overview
This project analyzes Netflix content data to uncover trends in:
- Content growth over years
- Movie vs TV show distribution
- Country-wise content availability
- Genre popularity
- Monthly content release patterns

## Tech Stack
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Jupyter Notebook

## Project Workflow
1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. Data transformation
4. Dashboard creation in Power BI
5. Insights extraction

## Key Insights
- Movies dominate the Netflix catalog
- Content growth accelerated after 2015
- Certain countries contribute significantly more content
- Drama and International genres appear frequently
